
# Project Base

Base padrão para iniciar projetos com planejamento, documentação, agentes, segurança, design, workflows e memória de projeto.

A Project Base existe para transformar uma ideia solta em uma base organizada de projeto, pronta para ser executada com ChatGPT, Claude, Codex ou outro agente de IA.

## Fluxo recomendado

1. Conversar primeiro com o ChatGPT para planejar o projeto.
2. Fazer uma entrevista detalhada sobre objetivo, público, páginas, dados, integrações, segurança e identidade visual.
3. Preencher os documentos iniciais.
4. Gerar comandos e prompts para Claude/Codex.
5. Executar a implementação com agentes.
6. Fazer o agente atualizar estado, progresso, decisões e relatórios.
7. Enviar os relatórios de volta ao ChatGPT para análise e próximos passos.

## Arquivos principais

- `START_HERE.md`
- `AGENTS.md`
- `CLAUDE.md`
- `CODEX.md`
- `docs/PROJECT_MAP.md`
- `docs/agent-state/CURRENT_SESSION.md`
- `docs/agent-reports/latest.md`

## Regra principal

A documentação deve estar viva. Se o projeto muda, os arquivos de memória, progresso, decisões e relatórios também devem mudar.
