## Inspirações iniciais da Project Base

### WorkLog

Tipo:
Projeto próprio / dashboard / acompanhamento / links compartilhados.

O que observar:
- Mensagens de pagamento;
- links compartilhados;
- atualizações para cliente;
- status visual;
- comprovantes;
- filtros;
- padrão de documentação.

O que adaptar:
- estrutura de acompanhamento;
- padrão de mensagem WhatsApp;
- status e feedbacks.

### CORE / CORE LAB

Tipo:
Projeto próprio / dashboard / identidade operacional.

O que observar:
- UUID interno;
- sidebar;
- identidade visual;
- modo claro/escuro;
- logo;
- preview de link;
- organização de docs.

O que adaptar:
- padrões visuais;
- estrutura de projeto;
- decisões arquiteturais.

### MazyOS / MazyUI

Tipo:
Inspiração externa / sistema de base / skills.

O que observar:
- ideia de skills;
- sistema base reutilizável;
- arquivos de orientação;
- fluxo para criar projetos.

O que adaptar:
- conceito de recursos auxiliares;
- organização de workflows;
- onboarding de uso.

O que evitar:
- copiar estrutura sem adaptar à identidade do usuário.
