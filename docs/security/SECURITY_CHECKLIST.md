
# Security Checklist

## Checklist básico

- [ ] Rotas privadas protegidas
- [ ] APIs validam permissão
- [ ] POST/PUT/DELETE protegidos
- [ ] RLS revisada
- [ ] Secrets fora do frontend
- [ ] Links públicos usam tokens seguros
- [ ] Webhooks validam assinatura
- [ ] Storage não expõe arquivos privados indevidamente
- [ ] Teste de acesso cruzado entre usuários/clientes
