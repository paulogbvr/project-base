
# Shared Link Security

## Regras

- Links compartilhados devem usar token curto, aleatório e não previsível.
- Nunca usar ID numérico sequencial em link sensível.
- Evitar expor UUID técnico em link público quando um token resolver.
- Permitir expiração/revogação quando necessário.
- Não colocar dados sensíveis no title, description, URL ou imagem de preview.
- Registrar acesso quando fizer sentido.
