
# Folder Structure

Estrutura sugerida para projetos Next.js.

src/
app/
components/
ui/
layout/
feedback/
integrations/
features/
lib/
supabase/
validations/
ids/
utils/
data/
styles/

## Regra

Componentes genéricos ficam em `components`.

Regras de negócio e módulos específicos ficam em `features`.

Integrações ficam em `lib`.
