# Share Messages

Este documento define mensagens de compartilhamento de links.

## Regra principal

Mensagens para WhatsApp/texto final devem seguir:

- `docs/brand/WHATSAPP_MESSAGE_STYLE.md`

## Quando usar

Usar mensagens prontas para:

- compartilhar link de projeto;
- enviar acompanhamento ao cliente;
- enviar cobrança;
- enviar comprovante;
- enviar proposta;
- enviar atualização;
- enviar pedido;
- enviar link de aprovação;
- enviar acesso a página pública.

## Estrutura recomendada

1. Título curto.
2. Contexto em citação.
3. Data/horário quando fizer sentido.
4. Campos principais.
5. Chamada para ação.
6. Link abaixo.

## Exemplo genérico

*→ Link de Acompanhamento!*

> *Projeto: →* Nome do Projeto
> _*08/06/2026 - 21:40*_

*Você pode acompanhar as atualizações pelo link abaixo:* ↓
https://exemplo.com/share/lk_token

## Exemplo de proposta

*→ Proposta Disponível!*

> *Projeto: →* Nome do Projeto
> _*Atualizado em 08/06/2026 - 21:40*_

*Resumo: →* A proposta já está disponível para análise.

*Acesse pelo link abaixo:* ↓
https://exemplo.com/proposta/lk_token

## Exemplo de atualização

*→ Atualização Registrada!*

> *Projeto: →* Nome do Projeto
> _*08/06/2026 - 21:40*_

*Status: →* Em andamento
*Etapa: →* Revisão visual

*Você pode acompanhar todas as atualizações pelo link abaixo:* ↓
https://exemplo.com/share/lk_token

## Regras

- Não usar `#` ou `##`.
- Não usar emoji por padrão.
- Usar `→` para campos.
- Usar `↓` antes do link.
- Usar `>` para contexto.
- Usar negrito/itálico no padrão WhatsApp.
- Não esconder link em Markdown.
- Não expor dados sensíveis.

## Checklist

- [ ] Mensagem segue estilo WhatsApp?
- [ ] Não usa título Markdown?
- [ ] Não usa emoji por padrão?
- [ ] Link está visível?
- [ ] Não há dados sensíveis?
- [ ] Existe chamada para ação?
