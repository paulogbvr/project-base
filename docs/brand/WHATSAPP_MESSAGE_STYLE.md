
# WhatsApp Message Style

Padrão oficial para mensagens finais em WhatsApp/texto.

## Permitido

- `*texto*` para negrito
- `_texto_` para itálico
- `_*texto*_` para negrito + itálico
- `>` para citação/destaque
- `→` para direção/campo
- `↓` para indicar continuação/link abaixo
- `•` para bullet simples

## Evitar

- Não usar `#` ou `##` como título
- Não usar emojis por padrão
- Não usar formatação de documento quando a mensagem for para WhatsApp
