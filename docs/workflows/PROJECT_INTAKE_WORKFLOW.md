
# Project Intake Workflow

Perguntas para iniciar um projeto.

## Perguntas essenciais

1. Qual é o tipo de projeto?
2. Quem é o cliente?
3. Qual problema resolve?
4. Quem usa?
5. Quem administra?
6. Quais páginas precisa?
7. Quais dados precisa guardar?
8. Vai ter login?
9. Vai ter pagamento?
10. Vai ter API externa?
11. Tem documentação da API?
12. Tem exemplos/referências?
13. Tem identidade visual?
14. Qual cor de acento?
15. Precisa modo claro/escuro?
16. Precisa link compartilhado?
17. Tem dados sensíveis?
18. Qual a entrega mínima?
19. Qual o prazo?
20. Qual o plano comercial?
