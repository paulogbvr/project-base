
# Feedback System

## Toasts

Toda ação importante deve gerar feedback visual.

Exemplos:

- copiar link
- salvar
- excluir
- enviar
- baixar
- testar conexão
- alterar status
- erro de validação
- upload concluído

## StatusPulse

Estados previstos:

- success / ok → verde pulsando
- warning / pending → amarelo pulsando
- error → vermelho pulsando
- neutral → cinza
