
# Sidebar Pattern

## Padrão

- Logo no topo
- Nome do projeto
- Hover na logo/sidebar
- Botão de expandir/recolher
- Ícone de sidebar adequado
- Estado ativo nos itens
- Perfil mockado no footer
- Persistência do estado aberto/recolhido
- Sem glitch de hidratação

## Ícones sugeridos

- PanelLeft
- PanelLeftClose
- ChevronLeft
- ChevronRight
