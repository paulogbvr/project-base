
# Design System

## Padrão visual base

- Monocromático premium por padrão
- Dark mode e light mode
- Cor de acento definida por projeto
- Se a cor não for informada, perguntar ao usuário
- Interface limpa, forte e organizada

## Componentes esperados

- Botões
- Cards
- Inputs
- Badges
- Status
- Toasts
- Skeletons
- Sidebar
- Header
