# Pricing

Este documento define padrões para planos, preços, condições comerciais e proposta.

## Objetivo

Registrar opções comerciais do projeto de forma clara.

Este arquivo pode ser usado para:

- proposta;
- apresentação;
- negociação;
- precificação futura;
- comparação de planos.

## Estrutura recomendada

Para cada plano, registrar:

- nome do plano;
- objetivo;
- implantação;
- mensalidade;
- forma de pagamento;
- o que inclui;
- o que não inclui;
- condição especial;
- validade da proposta.

## Formas de pagamento

Quando fizer sentido, apresentar:

### Pix / à vista

- menor valor;
- melhor condição;
- economia clara.

### Cartão

- valor maior;
- opção parcelada;
- mais flexível para caixa do cliente.

## Exemplo de estrutura

### Plano Essencial

Pix:

- Implantação: R$ 3.500
- Mensalidade: R$ 497/mês

Cartão:

- Implantação: R$ 4.200 em até 6x de R$ 700
- Mensalidade: R$ 497/mês

Economia no Pix:

- R$ 700

## Regra comercial

A opção Pix deve ser apresentada como melhor condição, não como desconto desesperado.

Frase sugerida:

A melhor condição é no Pix, porque reduz o custo inicial. Mas, se for melhor para o caixa da empresa, também existe a opção parcelada no cartão.

## Planos

Modelo:

### Plano 1 — Entrada

Para cliente que quer começar com o essencial.

### Plano 2 — Profissional

Melhor custo-benefício.

### Plano 3 — Completo / Multiunidade

Para operação maior, mais recursos e expansão.

## Recorrência

Mensalidade pode cobrir:

- hospedagem;
- suporte;
- pequenos ajustes;
- monitoramento;
- backups;
- manutenção;
- melhorias simples;
- atualizações de segurança.

Deixar claro o que não está incluso.

## Condição piloto

Quando for primeiro case ou projeto estratégico, pode existir condição especial.

Registrar:

- valor original;
- condição piloto;
- prazo da condição;
- contrapartida, se houver;
- possibilidade de usar como case.

## Checklist

- [ ] Planos definidos?
- [ ] Pix e cartão separados?
- [ ] Economia no Pix clara?
- [ ] Mensalidade definida?
- [ ] O que inclui está claro?
- [ ] O que não inclui está claro?
- [ ] Validade da proposta definida?
- [ ] Condição piloto registrada, se houver?
