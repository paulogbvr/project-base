# Proposals Skill

Skill auxiliar para criação de propostas comerciais, escopos, PDFs e apresentações.

Esta skill ajuda a transformar o planejamento do projeto em uma proposta clara para o cliente.

## Objetivo

Criar propostas que mostrem:

- problema;
- solução;
- escopo;
- diferenciais;
- fases;
- investimento;
- recorrência;
- prazos;
- próximos passos.

## Quando usar

Usar para:

- proposta de site;
- proposta de sistema;
- proposta de landing page;
- proposta de automação;
- proposta de consultoria;
- proposta de manutenção;
- proposta de MVP;
- proposta para cliente local;
- proposta para SaaS.

## Regra principal

Não vender apenas “site” ou “sistema”.

Vender resultado, clareza, operação, profissionalismo, controle e economia de tempo.

## Estrutura recomendada

1. Capa
2. Contexto
3. Problema identificado
4. Solução proposta
5. O que será entregue
6. O que não está incluso
7. Fases do projeto
8. Prazos
9. Investimento
10. Condições de pagamento
11. Mensalidade/suporte
12. Próximos passos

## Diagnóstico

A proposta deve mostrar que o problema foi entendido.

Exemplos:

- experiência atual ruim;
- operação manual;
- cliente não acompanha status;
- falta de link profissional;
- falta de controle;
- dependência de plataforma externa;
- retrabalho;
- baixa confiança visual.

## Escopo

Separar:

### Incluso

O que será feito.

### Não incluso

O que fica fora.

### Futuro

O que pode entrar depois.

## Preço

Quando houver Pix e cartão, apresentar separado.

Exemplo:

Pix:

- Implantação: R$ X
- Mensalidade: R$ Y/mês

Cartão:

- Implantação: R$ Z em até N vezes
- Mensalidade: R$ Y/mês

Mostrar economia no Pix quando houver.

## Tom

Tom recomendado:

- profissional;
- claro;
- direto;
- confiante;
- sem exagero;
- sem parecer informal demais.

## PDF

Quando gerar PDF:

- capa bonita;
- hierarquia clara;
- páginas curtas;
- visual consistente;
- dados do cliente;
- data;
- validade da proposta;
- contato.

## Apresentação ao vivo

Se for apresentar presencialmente:

- começar pelo problema;
- mostrar demo;
- mostrar benefício;
- só depois mostrar preço;
- fechar com próximo passo claro.
