# Google Ads Skill

Skill auxiliar para planejamento, criação e revisão de campanhas de Google Ads.

Esta skill não faz parte do sistema principal. Ela é um recurso para organizar campanhas, anúncios, palavras-chave, páginas de destino e acompanhamento.

## Objetivo

Ajudar a criar campanhas com:

- intenção clara;
- público correto;
- palavras-chave bem escolhidas;
- anúncios coerentes;
- página de destino alinhada;
- conversão mensurável;
- revisão antes de publicar.

## Quando usar

Usar quando o projeto envolver:

- tráfego pago;
- captação de leads;
- vendas;
- lançamento;
- campanha local;
- pesquisa de intenção;
- anúncios para WhatsApp;
- anúncios para landing page.

## Informações necessárias

Antes de criar campanha, levantar:

- produto/serviço;
- público;
- região;
- orçamento;
- objetivo;
- oferta;
- diferenciais;
- página de destino;
- CTA;
- concorrentes;
- termos proibidos;
- restrições legais ou de plataforma.

## Tipos de campanha

Possíveis:

- pesquisa;
- performance max;
- display;
- remarketing;
- YouTube;
- local;
- chamada;
- campanha para WhatsApp, quando aplicável.

## Estrutura de planejamento

Para cada campanha, definir:

- nome;
- objetivo;
- público;
- região;
- orçamento;
- estratégia de lance;
- grupos de anúncio;
- palavras-chave;
- negativas;
- anúncios;
- extensões/assets;
- conversões;
- página de destino.

## Palavras-chave

Separar:

- intenção alta;
- intenção média;
- pesquisa informativa;
- termos de marca;
- termos de concorrente, se permitido;
- negativas.

## Anúncios

Cada anúncio deve ter:

- títulos;
- descrições;
- CTA;
- diferencial;
- clareza;
- coerência com a página;
- sem promessas falsas.

## Revisão

Antes de publicar, revisar:

- ortografia;
- política da plataforma;
- link final;
- rastreamento;
- telefone/WhatsApp;
- conversões;
- orçamento;
- região;
- palavras negativas.

## Landing page

Conferir se a página tem:

- headline forte;
- prova/autoridade;
- CTA visível;
- carregamento rápido;
- mobile bom;
- formulário ou WhatsApp funcionando;
- política/termos quando necessário.

## Tracking

Eventos possíveis:

- ad_clicked
- landing_page_viewed
- lead_form_started
- lead_form_submitted
- whatsapp_clicked
- conversion_completed

Registrar em:

- `docs/analytics/TRACKING_EVENTS_SELECTED.md`
