# Copywriting Skill

Skill auxiliar para criação e revisão de textos.

## Objetivo

Criar textos claros, fortes e alinhados ao canal.

## Quando usar

Usar para:

- mensagens de WhatsApp;
- anúncios;
- landing pages;
- propostas;
- posts;
- carrosséis;
- e-mails;
- páginas de sistema;
- onboarding;
- mensagens de erro;
- CTAs.

## Regra principal

Texto bom deve ser claro antes de ser bonito.

## Antes de escrever

Levantar:

- objetivo;
- público;
- canal;
- tom;
- ação desejada;
- contexto;
- restrições;
- palavras obrigatórias;
- palavras proibidas.

## Estruturas úteis

### PAS

- Problema
- Agitação
- Solução

### AIDA

- Atenção
- Interesse
- Desejo
- Ação

### Antes/depois

- situação atual;
- transformação;
- próximo passo.

## Tom

Possíveis tons:

- direto;
- premium;
- emocional;
- técnico;
- institucional;
- comercial;
- consultivo;
- sensorial;
- operacional.

## Revisão

Todo texto deve passar por revisão de:

- ortografia;
- acentos;
- concordância;
- pontuação;
- espaçamento;
- clareza;
- tom;
- CTA.

## Erros recorrentes

Registrar em:

- `docs/project-memory/FINDINGS.md`

## Checklist

- [ ] Objetivo claro?
- [ ] Público claro?
- [ ] Canal correto?
- [ ] Tom correto?
- [ ] CTA claro?
- [ ] Ortografia revisada?
- [ ] Acentos revisados?
- [ ] Espaçamento revisado?
- [ ] Sem formato errado?
- [ ] Sem emoji quando padrão proíbe?
