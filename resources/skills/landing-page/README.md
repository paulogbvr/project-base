# Landing Page Skill

Skill auxiliar para criação de landing pages.

## Objetivo

Criar páginas com boa estrutura, clareza, conversão e visual profissional.

## Quando usar

Usar para:

- venda de serviço;
- captação de lead;
- apresentação de produto;
- campanha;
- lançamento;
- página institucional;
- página de proposta;
- página pública de sistema.

## Estrutura recomendada

1. Hero
2. Problema
3. Solução
4. Benefícios
5. Como funciona
6. Prova/autoridade
7. Oferta
8. FAQ
9. CTA final

## Hero

Deve conter:

- headline forte;
- subtítulo claro;
- CTA;
- visual principal;
- prova rápida quando houver.

## Problema

Mostrar que entende a dor.

Exemplos:

- perda de tempo;
- falta de organização;
- experiência ruim;
- retrabalho;
- baixa conversão;
- dependência de terceiros.

## Solução

Explicar de forma simples:

- o que é;
- para quem é;
- como resolve;
- por que é melhor.

## Benefícios

Traduzir recurso em resultado.

Exemplo:

Recurso: link compartilhado.  
Benefício: cliente acompanha sem precisar perguntar no WhatsApp.

## CTA

CTA deve ser claro.

Exemplos:

- Falar no WhatsApp
- Solicitar proposta
- Ver demonstração
- Começar agora
- Agendar apresentação

## Performance

Página deve:

- carregar rápido;
- funcionar no mobile;
- ter imagens otimizadas;
- evitar animações pesadas;
- ter CTA acessível.
